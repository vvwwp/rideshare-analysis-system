## 🔬 科学标准验证

### 1. 振动舒适性评价标准

#### 国际标准：ISO 2631-1:1997
- **标准全称**：Mechanical vibration and shock — Evaluation of human exposure to whole-body vibration — Part 1: General requirements
- **测量量**：频率加权RMS加速度 (m/s²)
- **频率范围**：0.5-80 Hz
- **加权函数**：Wk (垂直), Wd/Wf (水平)
- **舒适度阈值**：
  - < 0.315 m/s²：舒适
  - 0.315-0.63 m/s²：稍有不适
  - 0.5-1.0 m/s²：相当不舒适
  - 0.8-1.6 m/s²：不舒适
  - > 1.25 m/s²：极不舒适

#### 系统实现
```python
def _calculate_smoothness(self):
    # RMS加速度计算（符合ISO 2631-1）
    rms_acceleration = np.sqrt(np.mean(total_acceleration**2))
    
    # ISO 2631-1舒适度评价
    if rms_acceleration <= 0.315:
        score = 100
    elif rms_acceleration <= 0.63:
        score = 90 - (rms_acceleration - 0.315) / (0.63 - 0.315) * 20
    # ... 其他等级
```

### 2. 环境噪声评价标准

#### 世界卫生组织标准：WHO 2018
- **文档**：Environmental noise guidelines for the European Region
- **测量单位**：dB(A) - A加权分贝
- **道路交通噪声限值**：
  - Lden ≤ 53 dB(A)：日间平均
  - Lnight ≤ 45 dB(A)：夜间平均
- **健康影响评估**：基于暴露-反应关系

#### 系统实现

### 3. 车辆稳定性评价标准

#### 工程参考：Continental Automotive研究
- **数据来源**：Kontos et al. (2023), Vehicle Dynamics and Driver Behavior Analysis
- **测量指标**：角速度标准差 (rad/s)
- **驾驶风格分类**：
  - 冷静驾驶：σ ≤ 0.64 rad/s
  - 正常驾驶：σ ≤ 0.85 rad/s  
  - 激进驾驶：σ ≤ 1.24 rad/s
  - 危险驾驶：σ > 1.5 rad/s

#### 系统实现
```python
def _calculate_stability(self):
```

### 5. 价格性价比评价标准

#### 2023年武汉市出行成本调研
- **萝卜快跑（自动驾驶）**：1.2 元/km
- **网约车（滴滴）**：2.0-2.5 元/km
- **传统出租车**：2.5-3.0 元/km + 起步费

### 敏感度分析结果

| 指标 | 理论分数范围 | 敏感度 | 平衡因子 | 科学依据 |
|------|-------------|--------|----------|----------|

| 稳定性 | 5-100分 | 95 | 1.021 | Continental研究 |
| 噪声 | 5-100分 | 95 | 1.021 | WHO 2018指南 |
| 时间效率 | 5-100分 | 95 | 1.021 | 武汉交通局数据 |
| 价格 | 0-100分 | 100 | 0.97 | 武汉市场调研 |

### 数学公式验证

#### 熵权重计算
```
E_j = -1/ln(n) * Σ(p_ij * ln(p_ij))
W_j = (1 - E_j) / Σ(1 - E_i)
```

#### 平衡因子计算
```
标准敏感度 = Σ(敏感度_i) / n = 97
平衡因子_j = 标准敏感度 / 敏感度_j
```

#### 双层权重系统
```
最终权重_j = 平衡因子_j × 熵权重_j
归一化：最终权重_j = 最终权重_j / Σ(最终权重_i)
```

## 📋 数据收集规范

### Sensor Logger配置要求

#### 必需传感器
1. **加速度计（Accelerometer）**
   - 采样率：≥50 Hz
   - 量程：±8g
   - 用途：平顺性分析

2. **陀螺仪（Gyroscope）**
   - 采样率：≥50 Hz  
   - 量程：±1000 dps
   - 用途：稳定性分析

3. **GPS定位（Location）**
   - 采样率：1 Hz
   - 精度：≤5米
   - 用途：时间效率分析

4. **麦克风（Microphone）**
   - 采样率：44.1 kHz
   - 格式：WAV/MP4
   - 用途：噪声水平分析

#### 数据文件要求
- **CSV格式**：时间戳 + 传感器数值
- **命名规范**：Accelerometer.csv, Gyroscope.csv等
- **元数据**：Metadata.csv包含测试信息
- **标注文件**：Annotation.csv记录测试说明

### 测试规范
1. **手机固定**：使用车载支架，避免额外振动
2. **测试时长**：≥10分钟，获得统计意义
3. **路况记录**：在Annotation.csv中记录路况变化
4. **配对测试**：相同路线、相同时间段对比
